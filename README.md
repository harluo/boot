# 盘古 Golang应用程序框架

一个Golang应用程序快速开发框架，有以下特性

- 快速开发
- 内置配置文件加载（支持`Yaml`、`Toml`、`Json`以及`XML`）
- 内置强大的数据验证
- 强大的扩展功能
- 强大的配置功能
- 线程安全
- 内置依赖注入
- 语义化
