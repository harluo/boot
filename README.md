# 盘古 Golang应用程序框架

一个Golang应用程序快速开发框架，有以下特性

- 快速开发
- 内置配置文件加载
    - `Yaml`
    - `Toml`
    - `Json`
    - `XML`
- 内置强大的数据验证
- 强大的扩展功能
- 强大的配置功能
    - 徽标
    - 帮助信息
    - 命令行
    - 一切可配置点皆可配置
- 线程安全
- 内置依赖注入
- 语义化
    - 方便使用
    - 多态行为，同一个调用在不同的场景下有不同的意义
